# SantanderCodingTest

## How to run the application

### 1. To have a local version of the project, copy the github URL repository from github, paste the URL into the git terminal, and click Enter to clone the repository.

![imagen](https://github.com/luisjavierluna/SantCondingTest/assets/78492618/f4bcd240-6a1b-4c48-b25e-68d11a114a30)

![imagen](https://github.com/luisjavierluna/SantCondingTest/assets/78492618/c57ff7ec-1ffc-46e5-8be7-da9dac768ed2)

### 2. Open the project in Visual Studio and set up API project as Startup Project
![imagen](https://github.com/luisjavierluna/SantCondingTest/assets/78492618/6e0d921c-da20-4ad5-bb55-2ef4dd46fad0)

### 3. Start the api with any of the debugging buttons

![imagen](https://github.com/luisjavierluna/SantCondingTest/assets/78492618/740843df-902d-45f9-98e3-c9f389e95bc1)

### 4. The live server will be open automatically in the url relative to swagger. Here we will be able to test manually the api. You will see only one button "Get" with the enpoint /api/Stories.

![imagen](https://github.com/luisjavierluna/SantCondingTest/assets/78492618/d374c4c3-1010-4c5c-bd97-85bdfc94bbd5)
![imagen](https://github.com/luisjavierluna/SantCondingTest/assets/78492618/3996f17f-dd84-4096-b764-5822ffa82c74)

### 5. After clicking on "Get" button, an integer type parameter called storiesNumber is requested, enter the number and click to "Execute" button

![imagen](https://github.com/luisjavierluna/SantCondingTest/assets/78492618/a89669da-2783-4a99-8c58-0d1e6f5b4269)

### 6. You will receive a list of objects according to the integer you entered as a parameter. The objects will have the format required for the test.

![imagen](https://github.com/luisjavierluna/SantCondingTest/assets/78492618/938466f1-dc7d-4fbc-b3f0-8549c7128fb3)
